Chapter 2 — The Portable Workspace

Draft · 1,412 words · autosaved 12 minutes ago

2.1 What "portable" means here

Most productivity tools assume the user is the context: the same person sitting in front of the same machine, connected to the same accounts. Arasul inverts that — the drive is the context. It carries identity, credentials, notes, projects, and the AI that understands them.

When a user plugs the SSD into a friend's laptop, the machine becomes theirs for the session. When they eject, nothing remains. This is not a technical feature; it is a claim about what a personal computer can be.

2.2 The hardware contract

A USB-C SSD, formatted exFAT, with a vault-gated .boot/ directory and a user-readable content/ tree. Three cross-platform binaries — macOS, Windows, Linux — live at the drive root…
